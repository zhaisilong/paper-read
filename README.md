## 写在前面｜2026.02.26

过去一年，我们见证了多智能体（Multi-Agent）系统的迅猛发展。从工具增强到协作智能，Agent 正逐步进入科研核心流程。现在，是时候真正拥抱 ——

Agent for Science · Writing · Research

MolAstra 论文阅读项目，致力于系统化整理 AI4SCI（AI for Science）领域的前沿成果，构建一个 Human / Agent 双可读 的知识体系，让知识既服务于人类科研者，也服务于智能体协作系统。

我的核心目标包括：

- 📚 累积高质量 Markdown 论文笔记，按领域系统分类
- 🧠 构建 AI4SCI 论文分类学（Taxonomy）
- 🛠 设计覆盖 阅读 / 分析 / 写作 / 修改 等技能组合的多智能体工作流
- 🤝 打造人类 × Agent 协同的科研协作基础设施

目前，我们已在 MolAstra 累积 数百篇结构化 Markdown 论文笔记。

计划于 2026 年 6 月正式启动「智能科研计划」，探索面向未来的 Agent-native 科研范式。
